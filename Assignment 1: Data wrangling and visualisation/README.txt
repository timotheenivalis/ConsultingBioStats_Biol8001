In this assignment, you will demonstrate the skills you have learned in using R for simple data manipulation, using some new datasets. You will also learn some new R skills --  specifically, how to join different datasets together.

Please download the assignment template found below, and the data file.

The assignment template is an R markdown document. Please follow the instructions in that document. Add your R code in the chunks as required, then when you are finished, knit the document to html and upload the resulting file as your assignment submission. Please open your submission and read through it before submitting to ensure everything worked.
